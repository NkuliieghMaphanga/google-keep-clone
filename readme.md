#  Google Keep Clone

A responsive **Google Keep-inspired** web application built with **HTML5, CSS3, and Vanilla JavaScript**. This project allows users to create, organize, edit, archive, restore, search, and delete notes while storing data locally using the browser's **Local Storage API**.

---

##  Project Overview

This project recreates the core functionality of **Google Keep** as a front-end web application. It demonstrates modern web development concepts such as semantic HTML, responsive CSS, JavaScript DOM manipulation, CRUD operations, and Local Storage.

The application is designed to provide users with a simple and intuitive note-taking experience while showcasing clean code structure and responsive design principles.

---

##  Project Objectives

The objectives of this project were to:

- Build a fully responsive web application using HTML, CSS, and JavaScript.
- Apply semantic HTML for accessibility and maintainability.
- Implement CRUD (Create, Read, Update, Delete) functionality.
- Store application data using Local Storage.
- Create a modern, user-friendly interface inspired by Google Keep.
- Demonstrate clean coding practices and organized project structure.

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/yourusername/google-keep-clone.git
```

### Open the Project

Open the project folder in **Visual Studio Code**.

### Run the Application

You can either:

- Double-click `index.html`, or
- Use the **Live Server** extension in Visual Studio Code.

To use Live Server:

1. Install the Live Server extension.
2. Right-click on `index.html`.
3. Select **Open with Live Server**.

The application will open in your default browser.

##  Core Functionality

### Create Notes

Users can create notes by entering:

- Title
- Description
- Note color

Each note automatically includes the creation date.

### Edit Notes

Users can modify:

- Title
- Description
- Note color

### Archive Notes

Archived notes are removed from the main notes view while remaining accessible.

### Bin

Deleted notes are moved to the Bin where they can either be restored or permanently deleted.

### Search

Users can instantly search notes by title or content.

### Dark Mode

Users can switch between Light Mode and Dark Mode.

The selected theme is automatically saved.
